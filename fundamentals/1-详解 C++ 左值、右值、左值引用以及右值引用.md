## 1. 左值和右值

### 1.1 左值

在 C++ 中，左值（lvalue） 是表达式的一种值类别（value category），它表示一个具有身份（identity）且不能被移动的对象（或位域）。简单来说，左值指向一个明确的内存位置，你可以取得它的地址，并且它在表达式结束后依然存在。

**例如**:

```c++
//以下的a, p, *p, b都是左值
int a = 3;
int *p = &a;
*p;
const int b = 2;
```

#### 1.1.1 左值的核心特征
一个表达式是左值，当且仅当它具备以下特征:

1） **可以取地址:** 使用`&` 运算符能获得其内存地址

```c++
int x = 10;
&x;               // 合法，x 是左值
// &42;           // 错误，42 是右值，不能取地址
```

2）**有持久的状态：** 左值通常对应一个变量、函数、数组元素、数据成员等，其生命周期由作用域决定

3）**可以出现在赋值左边**（除非被 const 修饰）

```c++
const int c = 5;
// c = 10;        // 错误：c 是左值，但不可修改
```
4） **可以隐式转换为右值：** 当需要一个右值时，左值会自动进行左值到右值转换（例如作为算术运算符的操作数）


#### 1.1.2 左值的常见例子

表达式	是否为左值	说明
变量名 x	✅	最基本左值
数组下标 a[5]	✅	返回数组元素的左值
解引用 *ptr	✅	指针指向的对象是左值
对象成员 obj.m	✅	成员是左值（除非 m 是静态成员，返回左值）
字符串字面量 "hello"	✅	是左值，因为它存储在静态内存中
前置自增 ++x	✅	返回自增后的对象本身（左值）
后置自增 x++	❌	返回临时副本（右值）
函数返回左值引用 func()	✅	若返回类型为 T&
常量 42	❌	纯右值
临时对象 std::string("tmp")	❌	纯右值或将亡值
